# Exploration

Exploring the capabilities and strengths of vibe coding with Claude Opus 4.6. This repository serves as a testing ground for pushing the boundaries of what's possible when collaborating with AI on real software projects.

## Purpose

- Test how effectively Claude Opus 4.6 handles end-to-end development tasks
- Explore the strengths and limitations of vibe coding workflows
- Build projects of varying complexity to evaluate code quality, architecture decisions, and problem-solving ability
- Document observations and insights along the way

## What is Vibe Coding?

Vibe coding is a development approach where you describe what you want at a high level and let an AI assistant handle the implementation details. Rather than writing every line by hand, you guide the direction and the AI translates your intent into working code.

## Structure

Each experiment or project lives in its own directory, organized as they are added.
